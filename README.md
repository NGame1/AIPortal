# AI Portal 
AI Portal is a proxy between GitHub Copilot and your provider. 

## Using AI Portal
- Install the latest version of the extension from Releases
- Open your Visual Studio then navigate to Extensions -> AI Portal Settings
- Set your Base URL and API Key from your provider. (We tested OpenRouter and Z.Ai)
- After saving a command line window will pops uplike below
<img width="1483" height="762" alt="image" src="https://github.com/user-attachments/assets/df648cdf-1ee2-43a7-816e-2864626d08b1" />
- In your GitHub Copilot window, open Manage models dialog and the select your provider as Ollama
<img width="977" height="308" alt="image" src="https://github.com/user-attachments/assets/d16ea60b-f0ee-416c-92c1-a3f1d16d7987" />
- Click the add button then select your models and save
- In some cases, If you didn't see available models click on the Save or cancel button and try again, It will work.
- Now choose your model and ask everything or use agent modes and etc.
